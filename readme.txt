Paste images from clipboard in posts or comments with Clipboard Images.

== Description ==

* Author: CasePress
* Version: 0.2
* Author URI: https://github.com/casepress/WordPress-Image-Clipboard

Support paste images from clipboard for posts & comments (based on filereader.js). For example, 
you can take a screenshot using monosnap, and attach it to a post or comment using keyboard shortcuts CTRL+V.

== Installation ==

1.	Download the Clipboard Images archive in zip format and
	extract the files on your computer. 

2.	Extract Clipboard Images archive in the `wp-content/plugins`
	directory of your WordPress installation. 
	
3.	Enjoy it.
